# Data-Analyst-Task3
A comprehensive business intelligence project featuring interactive KPI tracking, customer segmentation analysis, and an automated dashboard built with Python, Plotly, and Streamlit.

Project Overview \n
This project performs end-to-end analysis on e-commerce sales data to extract meaningful business insights. Built with Python, it combines statistical analysis, data visualization, and interactive dashboarding to support data-driven decision making.
Dataset: sales_data.csv containing 1,000 transactions with customer demographics, product information, sales metrics, and satisfaction scores.\n
Key Business Questions Addressed:
Which customer segments drive the most revenue?
What is the impact of discounts on sales performance?
Which regions and channels perform best?
How do product categories compare in terms of returns and satisfaction?
What factors correlate with customer satisfaction?



Quick Start
Prerequisites
Python 3.8 or higher
pip package manager
Git (for cloning)
Installation


# 1. Clone the repository
git clone https://github.com/yourusername/sales-analytics-dashboard.git
cd sales-analytics-dashboard

# 2. Create and activate virtual environment (recommended)
python -m venv venv
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt


Running the Analysis
Option A: Jupyter Notebook (Exploratory Analysis)
jupyter notebook notebooks/analysis.ipynb
# → Execute all cells to generate KPIs, charts, and insights
